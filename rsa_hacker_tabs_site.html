RSA // HACKER CONSOLE

Две вкладки для кодирования и декодирования текста. Выберите шифр, задайте параметры RSA и работайте прямо в браузере.

root@matrix:~# init rsa_console --mode secure

Входные данные

Результат

Вывод формируется как последовательность чисел, разделённых пробелами. Каждый байт текста шифруется отдельно через RSA.

Входные данные

Результат

Для расшифровки используются те же параметры RSA: p, q и e. Приватная экспонента d вычисляется автоматически в браузере.
Важно: это демонстрационная реализация RSA для учебных целей. Для реальной криптографии не шифруют сообщение побайтно и используют безопасную генерацию ключей.
